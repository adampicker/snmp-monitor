About project
         Development of information technology has made that every enterprise needs computers. The increasing number of computers in companies creates necessity for responsible management.
         To deal with that responsibility and ease the companies network administrators work, author designed and implemented a project described in this thesis. System is created to suplement or be the only source of information about many working stations.
         Application provides comprehensive data about each working station with live updates. The application allows to customize which data is retrieving and present it with reactive and user friendy way.
         Application is also capable to present historical data for the purpose of creating reports and drawing conclusions. All assumptions of this project have been implemented and described. All functionalities of this system have been meticulously tested.
